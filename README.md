# InnovateX Hackathon Website

Landing website for **InnovateX Hackathon** — organized by **CSI SAHE** in collaboration with **GDG VRSEC**.

## Features

- Modern responsive landing page UI
- Registration CTA + downloadable PPT template link
- Event highlights + roadmap timeline

## Tech stack

- HTML, CSS, JavaScript

## Run locally

Option 1 (quickest):

- Open `index.html` in your browser.

Option 2 (recommended, local server):

- Use any static server (examples):
  - VS Code extension “Live Server”
  - Python: `python -m http.server 5173`
  - Node: `npx serve .`

Then open the shown local URL in your browser.

## Project structure (high level)

- `index.html`: landing page
- `style.css`: styles
- `script.js`: client-side behavior
- `assets/`: images/downloads (e.g., PPT template)
